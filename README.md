A computer vision pipeline that converts architectural floor plans into structured layouts that can later generate interactive 3D walkthrough environments.

**Pipeline components**
- Image preprocessing and normalization
- Wall and intersection detection
- Layout graph construction
- Room boundary reconstruction
- 3D model generation pipeline

**Tech stack:** Python, OpenCV, PyTorch, Three.js

</details>
